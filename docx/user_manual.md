# 高级无人机控制系统 - 用户说明书

## 1. 简介

欢迎使用 Project AirSim 高级无人机控制系统！本系统提供了一套完整的无人机仿真控制方案，让您可以在虚拟环境中体验无人机飞行、传感器数据采集和多种控制方式。

### 1.1 主要功能

- 🚁 **多机型支持**：四旋翼、六旋翼、倾斜旋翼(VTOL)三种无人机
- 📷 **双相机系统**：前视相机+下视相机，支持实时显示和录像
- 📸 **手动拍照**：随时对前视/下视相机进行独立拍照保存
- 🔦 **LiDAR点云**：实时3D点云可视化，支持快照保存
- 🎮 **键盘控制**：WASD+方向键，直观操控无人机
- 📡 **UDP控制**：接收外部程序发送的控制指令
- ✈️ **VTOL模式切换**：倾斜旋翼可在多旋翼/固定翼模式间切换
- 💾 **数据保存**：录像、拍照、点云、控制日志全部自动保存

## 2. 环境准备

### 2.1 系统要求

- 操作系统：Windows 10/11
- Python版本：3.11或更高
- Project AirSim仿真器：已安装并运行

### 2.2 安装依赖

在命令行中执行以下命令安装所需Python库：

```bash
pip install opencv-python numpy keyboard open3d
```

> 注意：keyboard库需要管理员权限才能正常工作

### 2.3 启动仿真器

在运行本系统之前，请确保 Project AirSim 仿真器已经启动并正在运行。

## 3. 快速开始

### 3.1 启动程序

1. 打开命令行（建议以管理员身份运行）
2. 进入脚本所在目录：
```bash
cd d:\projectairsim\ProjectAirSim\client\python\mine
```
3. 运行脚本：
```bash
python advanced_drone_control.py
```

### 3.2 选择无人机型号

程序启动后会显示以下菜单：

```
============================================================
       Project AirSim 高级无人机控制系统
============================================================

请选择无人机型号：
  1 - 四旋翼（灵活机动，适合近距作业）
  2 - 六旋翼（载重更大，冗余度更高）
  3 - 倾斜旋翼VTOL（支持固定翼/多旋翼切换，适合长距飞行）

请输入选项编号 (1/2/3):
```

输入对应数字选择机型：
- **1 - 四旋翼**：最常用的机型，灵活机动，适合近距离飞行和拍照
- **2 - 六旋翼**：比四旋翼更稳定，载重更大，有一个电机故障也能安全飞行
- **3 - 倾斜旋翼VTOL**：可以像直升机一样悬停，也可以像飞机一样前飞，适合长距离飞行

### 3.3 选择控制模式

```
请选择控制模式：
  1 - 手动控制（键盘WASD+方向键控制）
  2 - UDP自动控制（接收外部UDP指令控制）

请输入选项编号 (1/2):
```

- **1 - 手动控制**：使用键盘直接操控无人机（推荐新手使用）
- **2 - UDP自动控制**：由外部程序通过网络发送控制指令

### 3.4 起飞与飞行

选择完成后，程序会自动：
1. 连接到仿真器
2. 加载场景和无人机
3. 解锁并起飞

起飞完成后，您就可以开始控制无人机飞行了！

## 4. 键盘控制详解

### 4.1 基本飞行控制

| 按键 | 功能 | 说明 |
|------|------|------|
| W | 前进 | 无人机朝机头方向前进 |
| S | 后退 | 无人机朝机头方向后退 |
| A | 左移 | 无人机向左侧平移 |
| D | 右移 | 无人机向右侧平移 |
| ↑ (上箭头) | 上升 | 无人机向上爬升 |
| ↓ (下箭头) | 下降 | 无人机向下降落 |
| ← (左箭头) | 左转 | 无人机机头向左旋转 |
| → (右箭头) | 右转 | 无人机机头向右旋转 |

> 提示：所有方向都是相对于无人机机头方向的。就像你站在无人机上看到的方向一样。

### 4.2 速度调节

| 按键 | 功能 | 说明 |
|------|------|------|
| + 或 = | 加速 | 每次增加1 m/s，最大20 m/s |
| - | 减速 | 每次减少1 m/s，最小1 m/s |

当前飞行速度会在控制台显示。

### 4.3 相机操作

| 按键 | 功能 | 说明 |
|------|------|------|
| F | 前视相机拍照 | 保存前视相机当前画面为图片 |
| G | 下视相机拍照 | 保存下视相机当前画面为图片 |

> 相机在程序运行期间会持续录像，按F/G键是在录像基础上额外保存单张高清照片。

### 4.4 LiDAR操作

| 按键 | 功能 | 说明 |
|------|------|------|
| L | 保存点云快照 | 将当前LiDAR扫描的点云数据保存到文件 |

> LiDAR点云会实时显示在单独的3D可视化窗口中，按L键可以保存当前时刻的点云数据用于后续分析。

### 4.5 VTOL模式切换（仅倾斜旋翼）

| 按键 | 功能 | 说明 |
|------|------|------|
| V | 切换飞行模式 | 在多旋翼模式和固定翼模式之间切换 |

- **多旋翼模式**：像直升机一样悬停和低速飞行，适合起降和近距离作业
- **固定翼模式**：像飞机一样前飞，速度更快、效率更高，适合长距离飞行

> ⚠️ 切换到固定翼模式时，请确保无人机有足够的前飞速度，否则可能失速下坠！

### 4.6 退出控制

| 按键 | 功能 | 说明 |
|------|------|------|
| T | 着陆并退出 | 无人机自动着陆后退出程序 |
| Q | 直接退出 | 立即退出程序（会尝试着陆） |

## 5. UDP控制模式

### 5.1 概述

UDP控制模式允许外部程序通过网络发送控制指令来操控无人机。适用于自动驾驶算法测试、编队飞行等场景。

### 5.2 默认配置

| 参数 | 默认值 | 说明 |
|------|--------|------|
| 监听IP | 127.0.0.1 | 本机回环地址 |
| 监听端口 | 9876 | UDP端口号 |
| 接收周期 | 100ms | 每100ms接收一次指令 |

### 5.3 指令格式

UDP指令为JSON格式字符串，通过UTF-8编码发送。支持以下字段：

```json
{
  "velocity": {"vx": 2.0, "vy": 1.0, "vz": 0.0},
  "position": {"x": 10.0, "y": 5.0, "z": -3.0},
  "yaw_rate": 0.0
}
```

**字段说明**：

| 字段 | 类型 | 说明 |
|------|------|------|
| velocity.vx | float | 北向速度（m/s），正值朝北 |
| velocity.vy | float | 东向速度（m/s），正值朝东 |
| velocity.vz | float | 下向速度（m/s），正值朝下 |
| position.x | float | 目标位置X坐标（米，北向） |
| position.y | float | 目标位置Y坐标（米，东向） |
| position.z | float | 目标位置Z坐标（米，向下为正） |
| yaw_rate | float | 偏航角速度（度/秒） |

**优先级**：如果同时包含velocity和position，优先使用velocity。

### 5.4 Python发送示例

```python
import socket
import json

# 创建UDP套接字
sock = socket.socket(socket.AF_INET, socket.SOCK_DGRAM)
server_address = ("127.0.0.1", 9876)

# 发送速度控制指令
command = {
    "velocity": {"vx": 3.0, "vy": 0.0, "vz": 0.0},
    "yaw_rate": 0.0
}
data = json.dumps(command).encode("utf-8")
sock.sendto(data, server_address)

# 发送位置控制指令
command = {
    "position": {"x": 10.0, "y": 5.0, "z": -5.0}
}
data = json.dumps(command).encode("utf-8")
sock.sendto(data, server_address)

sock.close()
```

### 5.5 命令行参数

可以通过命令行参数修改UDP配置：

```bash
python advanced_drone_control.py --udp-ip 0.0.0.0 --udp-port 9999
```

| 参数 | 说明 | 默认值 |
|------|------|--------|
| --udp-ip | UDP监听IP | 127.0.0.1 |
| --udp-port | UDP监听端口 | 9876 |
| --address | 仿真器IP | 127.0.0.1 |
| --topicsport | Topic端口 | 8989 |
| --servicesport | Service端口 | 8990 |

## 6. 数据保存说明

### 6.1 保存位置

所有数据保存在 `drone_data` 目录下，每次运行创建一个以时间戳命名的文件夹：

```
drone_data/
└── 20260425_143022/          ← 运行时间：2026年4月25日 14:30:22
    ├── images/                ← 拍照图片
    ├── videos/                ← 录像文件
    ├── lidar/                 ← LiDAR点云数据
    └── logs/                  ← 控制日志
```

### 6.2 文件类型

| 类型 | 目录 | 格式 | 说明 |
|------|------|------|------|
| 前视录像 | videos/ | .avi | 前视相机持续录像 |
| 下视录像 | videos/ | .avi | 下视相机持续录像 |
| 前视拍照 | images/ | .jpg | 按F键手动拍照 |
| 下视拍照 | images/ | .jpg | 按G键手动拍照 |
| LiDAR点云 | lidar/ | .npy | 按L键保存的点云数据 |
| 控制日志 | logs/ | .csv | 所有控制指令记录 |

### 6.3 查看点云数据

LiDAR点云保存为NumPy的.npy格式，可以用Python读取和查看：

```python
import numpy as np
import open3d as o3d

# 加载点云
points = np.load("drone_data/20260425_143022/lidar/lidar_0001_xxx.npy")

# 使用Open3D可视化
pcd = o3d.geometry.PointCloud()
pcd.points = o3d.utility.Vector3dVector(points)
o3d.visualization.draw_geometries([pcd])
```

### 6.4 查看控制日志

控制日志为CSV格式，可以用Excel或文本编辑器打开：

```
timestamp,mode,vx,vy,vz,yaw_rate,pos_x,pos_y,pos_z
2026-04-25T14:30:25,manual,5.0,0.0,0.0,0.0,-1.0,8.0,-4.0
2026-04-25T14:30:26,udp,2.0,1.0,0.0,0.0,-0.5,8.5,-4.0
```

| 字段 | 说明 |
|------|------|
| timestamp | 操作时间 |
| mode | 控制模式（manual=手动，udp=自动） |
| vx, vy, vz | 三轴速度 |
| yaw_rate | 偏航角速度 |
| pos_x, pos_y, pos_z | 无人机位置 |

## 8. 常见问题

### Q1: 程序启动后无法连接仿真器

**A:** 请检查：
1. Project AirSim仿真器是否已启动
2. 仿真器IP地址是否正确（默认127.0.0.1）
3. 端口号是否匹配（默认topics=8989, services=8990）

### Q2: 键盘按键没有反应

**A:** keyboard库需要管理员权限：
1. 右键点击命令行，选择"以管理员身份运行"
2. 然后重新运行脚本

### Q3: 相机画面黑屏

**A:** 可能原因：
1. 无人机还在地面，相机被遮挡 → 先起飞再查看
2. 场景光照不足 → 更换场景或调整时间
3. 传感器配置错误 → 检查配置文件中相机的enabled字段

### Q4: LiDAR窗口没有显示

**A:** LiDAR可视化依赖Open3D库：
1. 确认已安装：`pip install open3d`
2. 如果是远程桌面连接，可能不支持3D渲染

### Q5: 录像文件无法播放

**A:** 录像使用XVID编码的AVI格式：
1. 使用VLC播放器打开
2. 或安装XVID解码器

### Q6: 倾斜旋翼切换固定翼模式后下坠

**A:** 切换到固定翼模式前需要保持足够的前飞速度：
1. 先用W键加速到至少5 m/s
2. 然后按V键切换到固定翼模式
3. 切换后继续保持前飞速度

### Q7: UDP模式收不到指令

**A:** 请检查：
1. 发送端IP和端口是否正确
2. 防火墙是否放行了UDP端口
3. 指令格式是否为合法JSON
4. 使用`--udp-ip 0.0.0.0`监听所有网络接口

## 10. 安全提示

1. ⚠️ 仿真环境中的操作不会影响真实无人机，请放心尝试
2. ⚠️ 切换VTOL模式前确保有足够前飞速度
3. ⚠️ 长时间录像会占用较多磁盘空间，注意存储容量
4. ⚠️ UDP模式下无指令时无人机会悬停，但不会自动避障
5. ⚠️ 程序异常退出时，数据可能未完整保存
